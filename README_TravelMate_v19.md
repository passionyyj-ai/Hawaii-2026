# TravelMate AI v19.0

## 추가 기능
- 날짜별 일정 추가
- 기존 일정 수정
- 일정 삭제 및 확인창
- 추가·수정·삭제 내역 기기 내 자동 저장
- 새로 추가한 일정에도 지도, 완료 체크, 첨부·메모 기능 연결
- 기존 무한 새로고침 방지 구조 유지

## 클라우드 동기화
현재 v19.0의 일정 변경과 첨부파일은 기기 내부에 저장됩니다. 다른 휴대폰과 동기화하려면 Supabase 프로젝트 URL과 anon public key를 연결하는 다음 단계가 필요합니다. service_role key는 사용하지 마세요.
