# TravelMate AI v20.4

## iPhone 홈 화면 음성통역 변경

- Safari 탭에서는 기존 음성 인식을 그대로 사용합니다.
- 홈 화면 아이콘(PWA/standalone)에서 마이크 버튼을 누르면 Safari의 통역 화면을 새 창으로 엽니다.
- Safari로 열린 페이지는 통역 탭을 자동 선택합니다.
- 홈 화면에서 반복되던 `aborted`, 무응답, 즉시 종료 문제를 우회합니다.
- 일정·예약·여행 공유·첨부파일 기능은 홈 화면 앱에서 계속 사용합니다.

## 설치

압축을 풀어 GitHub 저장소의 기존 파일을 모두 덮어쓴 후 Commit 합니다. 기존 홈 화면 앱은 완전히 종료한 다음 다시 실행합니다.
